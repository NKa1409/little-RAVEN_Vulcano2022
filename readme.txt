Raw data for the results presented in:

Karbach, N., Bobrowski, N. & Hoffmann, T. 
Observing volcanoes with drones: studies of volcanic plume chemistry with ultralight sensor systems. Sci Rep 12, 17890 (2022)

DOI: https://doi.org/10.1038/s41598-022-21935-5


"flight data" folder contains the data for: measurement flight 1 & 2 as well as the validation flight
"photoakkustic sensor" folder contains the data for the testing of the photoaccustic SCD41 sensor during a lab test and a test flight on vulcano.
